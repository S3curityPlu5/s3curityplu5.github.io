# mezmedicimedia.com

This is the GitHub Pages website for Mez Media.

## About

Mez Media provides creative media solutions including video production, photography, graphic design, and more.

## Setup

This site is hosted on GitHub Pages. To deploy:

1. Push this repository to GitHub.
2. Go to repository Settings > Pages.
3. Select the main branch as the source.
4. If using a custom domain, add `mezmedicimedia.com` to the Custom domain field and create a CNAME file.
